# EchoRoom Manager

总结下来manager的职责如下：

* 协助Host：
  * 负责session的话题安排（譬如今天调整host顺序呀什么的）；
  * 提醒host及时发话题；
  * 现场host需要帮助的时候，给与救援（譬如host迟到、没有准备充分帮忙控下场子）；
  * 对录音进行统一的整理；
* 邀请Guest：
  * 有小伙伴请假的时候，按照顺序邀请guest来体验；
  * 给Guest发送录音、收集Guest反馈；
* Echo together：
  * 参与大组活动的组织（optional）；
  * 活动现场带领小伙伴为自己的小组火拼：）
